#StreamUnpack

##A single-file library for decoding CBOR messages. 

__This is a work-in-progress and will likely be significantly changed in the near future. It is far from production-usable.__

Most CBOR libraries have limited support for dynamic content, assuming you know the type and content ahead of time. This is not useable in cases where contents can change significantly.

Currently, data is stored in a custom struct, this is largely temporary code thrown together for testing, so it will not be consistent with the completed version. Documentation will be provided once ready.
